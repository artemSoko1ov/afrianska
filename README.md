# Afrianska

Проект "Afrianska" - это современное веб-приложение, созданное с использованием Vite и современных веб-технологий.

## Технологии

- **Vite** - сборщик проекта и инструмент для разработки
- **ESLint** - линтер для проверки качества кода
- **Sass** - препроцессор CSS
- **PostCSS** с плагинами:
    - postcss-preset-env - позволяет использовать современные возможности CSS
    - postcss-pxtorem - преобразует пиксели в rem единицы

## Установка и запуск

### Предварительные требования

Убедитесь, что у вас установлен Node.js (версия 14 или выше)

### Установка зависимостей

```bash
npm install
```

### Запуск в режиме разработки

```bash
npm run dev
```

Приложение будет доступно по адресу: `http://localhost:5173`

### Сборка для production

```bash
npm run build
```

Собранные файлы будут размещены в директории `dist/`

### Просмотр собранного проекта

```bash
npm run preview
```

### Проверка кода линтером

```bash
npm run lint
```

### Автоматическое исправление ошибок линтера

```bash
npm run lint:fix
```

## Особенности конфигурации

- Проект использует кастомную конфигурацию ESLint от `@delement/eslint-config-master`
- Автоматическое преобразование px в rem через PostCSS
- Поддержка современных возможностей CSS через postcss-preset-env
